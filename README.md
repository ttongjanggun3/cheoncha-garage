# 🏎️ 천차 Garage (CheonCha Garage) 🏎️
# 제출물 1형태
## 한 문장 설명

자동차 제작 동아리의 출석부 데이터를 자동으로 계산하여 총출석, 부족 횟수, 초과 횟수를 정리해주는 출석 관리 자동화 프로그램

---

## 제작 목적

천차많별 동아리는 자동차 제작 대회를 준비하는 동아리로, 여러 명의 임원단과 부원들이 매주 작업에 참여한다. 기존에는 출석 관리를 주차별 엑셀 파일로 따로 작성하고 공유하는 방식으로 진행했다.

하지만 매주 다른 엑셀 파일이 올라오다 보니 이전 주차와 합산해서 확인하기 어렵고, 누가 몇 번 출석했는지 한눈에 보기 힘들었다. 특히 출석 횟수가 부족한 사람이나 기준보다 초과해서 출석한 사람을 매번 직접 계산해야 해서 관리 시간이 많이 들었다.

이러한 문제를 해결하기 위해 Notion에 주차별 출석부를 만들고, Python 프로그램을 이용해 출석 데이터를 자동으로 계산하는 프로그램을 제작하고자 한다. 현재 버전에서는 동아리 운영에 필요한 여러 기능 중 가장 실제로 필요하고 바로 사용할 수 있는 **출석 관리 자동화 기능**에 집중하였다.

---

## 초기 기획과 방향 수정

초기에는 천차 Garage를 출석, 일정, 작업 기록, 지식 공유를 모두 포함하는 협업 보조 시스템으로 기획하였다. 그러나 프로젝트를 진행하면서 한 학기 안에 모든 기능을 구현하기에는 범위가 너무 넓다고 판단하였다.

따라서 현재 버전에서는 가장 핵심적이고 실제 동아리 운영에 바로 사용할 수 있는 **출석 관리 기능**만 구현하는 방향으로 범위를 줄였다. 기존 초기 기획안은 별도의 파일로 옮겨 보관하였고, 현재 README에서는 출석 관리 자동화 기능을 중심으로 정리한다.

---

## 핵심 기능 3가지

### 1. 출석부 데이터 불러오기

- Notion에 작성된 주차별 출석부 데이터 불러오기
- 여러 주차 출석부 데이터 사용 가능
- Notion API를 활용하여 출석부 데이터베이스에 접근

### 2. 출석 횟수 자동 계산

- 사람별 총출석 횟수 계산
- 여러 주차에 흩어진 같은 이름의 출석 횟수 합산
- `월`, `화`, `수`, `금A`, `토B`, `청소`, `노션` 등 출석으로 인정되는 체크박스 항목 계산

### 3. 부족/초과 횟수 및 결과 파일 생성

- 기준 출석 횟수 대비 부족 횟수 계산
- 기준 출석 횟수 대비 초과 횟수 계산
- 부족 횟수가 큰 사람을 경고 대상자로 분류
- 전체 결과를 엑셀 파일로 저장
- 경고 대상자를 텍스트 파일로 저장
- 출석 현황을 HTML 파일로 저장

## 결과물 형태

Python 기반 출석 관리 자동화 프로그램

실행 후 생성되는 결과물은 다음과 같다.

- `출석결과.xlsx`  
  전체 부원의 총출석, 부족 횟수, 초과 횟수를 정리한 엑셀 파일

- `경고공지.txt`  
  부족 횟수가 큰 사람을 정리한 공지용 텍스트 파일

- `출석현황.html`  
  출석 결과를 웹페이지처럼 확인할 수 있는 HTML 파일
