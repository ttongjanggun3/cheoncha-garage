# 🏎️천차 Garage (CheonCha Garage)🏎️

## 한 문장 설명
자동차 제작 동아리의 출석, 일정, 작업 기록, 지식 공유를 통합 관리하는 협업 보조 시스템

---
## 제작 목적
천차많별 동아리는 자동차 제작 대회를 준비하는 동아리로 여러 명의 임원단과 부원들이 작업을 진행한다. 현재는 출석 관리, 작업 기록, 일정 공유가 카카오톡과 엑셀 파일 중심으로 이루어지고 있어 부원들이 기록이 쉽게 묻히거나 관리가 어려운 문제가 있다.

또한 설계, 노칭, 용접 등 각 파트의 경험과 지식이 체계적으로 정리되지 않아 후배 인수인계에도 어려움이 존재한다. 특히 임원단이 약 20명 정도 되다 보니 회의 날짜를 잡거나 전체 일정을 공유하는 과정도 비효율적이다.

이러한 문제를 해결하기 위해 동아리 운영에 특화된 협업 시스템을 기획하고, 기존 협업 도구(Notion)의 한계를 Python 프로그램으로 보완해보고자 한다.

---
## 핵심 기능 3가지
### 1. 출석 관리 시스템
- 출석 체크 기능
- 출석률 자동 계산
- 출석률 저조자 자동 표시
### 2. 일정 공유 및 회의 시간 추천 기능
- 동아리 일정 캘린더
- 개인 일정 등록
- 임원단 캘린더 일정 자동 비교 후 공통 가능 시간 추천
### 3. 작업 기록 및 지식 공유 기능
- 작업 기록 저장
- 파트별 작업 내용 공유
- 설계/용접/노칭 노하우 아카이브화
